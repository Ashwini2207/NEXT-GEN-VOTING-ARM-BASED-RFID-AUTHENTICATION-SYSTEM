# Next Generation Voting System Using ARM Based RFID Authentication

## Overview

A smart electronic voting system developed using ARM7 microcontroller and RFID technology to provide secure voter authentication and efficient vote management.

## Problem Statement

Traditional voting systems are prone to manual errors, impersonation, and time-consuming verification processes.

## Solution

Developed an RFID-based voting system that authenticates voters using RFID cards and allows secure electronic voting through an ARM7 microcontroller.

## Features

* RFID Based Voter Authentication
* Secure Electronic Voting
* Fast Voter Verification
* LCD Display Interface
* Unauthorized Access Prevention
* Real-Time Vote Recording

## Hardware Components

* LPC2129 ARM7 Microcontroller
* RFID Reader Module
* RFID Cards
* 16x2 LCD Display
* Push Buttons
* Power Supply

## Software Tools

* Embedded C
* Keil uVision
* Flash Magic

## Implementation

* RFID card is scanned for voter authentication.
* ARM7 validates the RFID information.
* Authorized voters are allowed to cast votes.
* Votes are recorded electronically.
* LCD displays authentication and voting status.

## Concepts Used

* Embedded C Programming
* ARM7 Microcontroller
* RFID Communication
* LCD Interfacing
* Digital Input Processing
* Embedded System Design

## Applications

* College Elections
* Organization Elections
* Secure Voting Systems
* Smart Governance Applications
